# EC2 Networking Troubleshooting Lab

## Overview
This project demonstrates troubleshooting AWS EC2 instance connectivity issues and networking configurations.

## AWS Services Used
- Amazon EC2
- Security Groups
- VPC (optional if applicable)

## Steps
1. Launched an EC2 instance using a Linux AMI
2. Configured security groups to allow SSH and HTTP access
3. Diagnosed public IP connectivity issues
4. Fixed SSH key pair permission errors
5. Tested remote access and confirmed connectivity

## Key Learnings
- How security groups affect EC2 connectivity
- Common SSH troubleshooting steps
- Public vs private IP understanding
